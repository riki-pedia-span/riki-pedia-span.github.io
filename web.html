Cómo es ser un desarrollador web

Hay muchas herramientas y atajos para el desarrollo web, pero no todos funcionan.

Por ejemplo, hay WordPress, un creador de sitios web gratuito* donde puede usar plantillas, crear archivos e incluso alojar un sitio web. También hay GitHub, un servicio gratuito de alojamiento de sitios web, donde alojo Rikipedia. Pero, ¿por qué estoy hablando de anfitriones? Necesita saber sobre el propio code. Podrías usar muchos lenguajes de código para hacer muchas cosas diferentes. Uso html para hacer mis sitios web, pero hay mucho más, incluidos javascript, ruby, python, php y muchos más. HTML es un buen lugar para comenzar porque es fácil de aprender y de depurar.
Aprendí a usar HTML en Kahn Academy, que también es donde pruebo mi código antes de cargarlo.

Hay muchos otros servicios y aplicaciones para probar, pero quédate con lo que te gusta.

*Mayormente gratis

Cómo hago mi sitio web

Empiezo ejecutando y escribiendo code a través de Khan Academy, para verificar si funciona.

Luego, tomo las fotos con mi teléfono y las subo a Los Riki-Docs.

Entonces yo ejecútelo a través de mi línea de comandos usando comandos como new-item [ruta y nombre del archivo.html] y set-content"[el código HTML]"

Olvidé la ruta jajaja

Luego la publico usando GitHub (el enlace lleva a la versión sin formato de la página de github).


Una vez que termino de cargar, agrego nuevos elementos a mi página de búsqueda personalizada de Google (la barra de búsqueda en la página de inicio). Luego cargo los archivos y espero a que se conviertan en Esto .

El enlace llamado "esto" lleva a la página de inicio
